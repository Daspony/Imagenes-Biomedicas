# Directrices para Claude

## Estilo de código

- **Sin emojis**: Nunca usar emojis en prints, comentarios, markdown ni en ningún lugar del código
- Usar mensajes claros y concisos con prefijos como [OK], [ERROR], [INFO], [WARNING]
- Mantener consistencia en el idioma (español para este proyecto)

## Estructura del proyecto

- Los notebooks están en `notebooks/` numerados del 00 al 06
- Los módulos reutilizables están en `utils/`
- Los datos de LUNA16 van en `LUNA16/`
